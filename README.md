## 🌀 About

**entropy-garden** is a living archive of experiments in data, artificial intelligence, and emergent systems.

This is not just a collection of notebooks.
It’s a **cultivation space** — where raw, chaotic inputs are transformed into structure, insight, and unexpected patterns.

```
input → noise → exploration → structure → signal
```

---

## 🧠 Philosophy

```
data ≈ entropy
models ≈ compression
insight ≈ emergence
```

* chaos is the starting point
* curiosity is the engine
* iteration is the path
* ambiguity is accepted
